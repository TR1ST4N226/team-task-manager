## ✨ Features

### 🎯 Task Management
- ✅ Create, Read, Update, Delete (CRUD) tasks
- 🏷️ Assign tasks to team members (single or multiple)
- 🎨 Set priorities: Low, Medium, High, Urgent
- 📊 Track status: To-Do, In Progress, Review, Done
- 📅 Due dates with visual deadline indicators
- 📎 File attachments
- 💬 Comments and task updates

### 👥 Team Collaboration
- 👤 Create and manage teams
- 📧 Invite members via email
- 🔑 Assign roles: Admin, Manager, Member
- 👀 Track who created/modified what and when

### 📊 Analytics & Reporting
- 📈 Dashboard with real-time metrics
- 📉 Task completion rate charts
- 📊 Team productivity trends
- 👤 Individual member performance tracking
- 📄 Export reports (PDF, Excel)

### 🎨 User Experience
- 🌓 Dark/Light mode
- 📱 Fully responsive (mobile, tablet, desktop)
- 🖱️ Drag-and-drop Kanban board
- 🔄 Real-time updates
- ♿ Accessibility features (ARIA labels, keyboard navigation)
